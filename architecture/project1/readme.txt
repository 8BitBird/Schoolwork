Python version 2.7
Program will disassemble 32 bit Stumpyleg v8 (see Zybooks Computer Architecture Ch.2) from a text file. Each 32 bit command must be on its own line with no whitespace. 
Program requries 2 command line arguments, -i <input_bin_file.txt> and -o <disassembled_output_file>
